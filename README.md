# Health Score Dashboard

## Description

The Health Score Dashboard is designed to provide holistic, general, evidence-based guidance for living a well-balanced and healthy lifestyle. Its purpose is to synthesise widely accepted, institutional-level recommendations—such as government and public health guidance—into a single, practical tool. By focusing on the most impactful habits, the app helps users cut through influencer-driven trends and optimisation culture that overemphasise niche interventions and can create unnecessary stress. This tool aims to empower users to prioritise behaviours that reliably improve energy, resilience, physical function, mental wellness, and long-term health.

---

## Features

- **Four Key Categories:** Recovery, Nutrition, Activity, and Mental wellness, each with a weighted contribution to overall health.  
- **Self-Assessment Scoring:** Users assign scores to behaviours based on their adherence to guidelines.  
- **Weighted Calculations:** Automatic aggregation of individual factors into category scores and an overall health score.  
- **Evidence-Based Guidance:** Provides practical reference points for optimal and 80/20 levels for each behaviour.  
- **Interactive Tables:** Enter and update scores easily; all calculations update in real time.  
- **Tooltips & Commentary:** Concise explanations for each factor and category, highlighting why it matters.  

---

## How to Use

1. Download the `index.html` file to your computer.  
2. Ensure the file is saved with the `.html` extension.  
3. Double-click the file to open it in your default web browser.  
4. Navigate between the tabs (Recovery, Nutrition, Activity, Mental) and input self-scores to see weighted category and overall scores update in real time.


## Notes

- This app provides general educational guidance and is **not personalised medical advice**.  
- Individual needs may vary; consult a healthcare professional for personal guidance.  
- Scores are self-reported and intended to help users prioritise meaningful behaviours.
